# Serverless API using AWS Lambda (Python)

## Project Overview

This project demonstrates a simple serverless API using AWS Lambda concepts and Python.

The function accepts an event input and returns a JSON response.

## Features

- Event-driven architecture
- Serverless computing concept
- JSON request processing
- JSON response generation

## Technologies Used

- Python
- JSON
- AWS Lambda (Concept)

## Files

- lambda_function.py
- test.py
- event.json
- response.json
- requirements.txt
